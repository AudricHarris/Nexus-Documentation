What is Nexus?

Nexus is an opinionated, modern systems programming language that aims to deliver:

  • Performance close to C++ / Rust
  • Much simpler syntax and learning curve
  • Strong type system with no null / undefined
  • First-class async/await and structured concurrency
  • Excellent compiler errors and developer experience
