Installation

Download the latest compiler executable from GitHub. Current recommended version: v1.3

  • • Download from Releases (recommended — check for v1.3 or newer)
  • • Or browse older versions in /versions folder
  • • Unzip → add to PATH (or run directly) → verify with nexus --version
  • • Coming soon: one-line installer script
After installing, try: 
nexus new my-project
cd my-project
nexus run
