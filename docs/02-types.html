Type Overview

A tour of Nexus types, initializing, and memory semantics.

Primitive Types

Nexus is statically typed. All primitives have explicit sizes. There is no implicit numeric coercion.

TypeDescriptionExample
i3232-bit signed integeri32 x = 42;
i6464-bit signed integeri64 big = 9999999999;
f3232-bit floatf32 pi = 3.14;
f6464-bit floatf64 e = 2.718281828;
boolBooleanbool flag = true;
strUTF-8 stringstr name = "Nexus";

Variables & Assignment

Variables are declared with their type (No null declaration). Nexus distinguishes between copy assignment (=) and explicit move (<-), giving you control over ownership at every assignment site.

copy vs move
i32 x = 10;               // copy
i32 y <- x;              // explicit move — x is no longer valid

i32 tmp &= y;  // borrow reference
Variable DeclarationSemantics
T x = valueCopy assignment
T x <- valueExplicit move (transfers ownership)
T x &= valueBorrow (reference, not ownership)

Enumerations (Enums)

Enums represent discrete choices or variants. They can carry auto-incremented variants, explicit mappings, or even wrap generic sub-types like Option<T>.

Note: You can directly evaluate or compare enum choices against raw integer constants (e.g., matching or comparing with 1).
enum declarations
enum Option<T>
{
    None,
    Some(T value)
}

enum Code
{
    A, // 1
    B, // 2
    C  // 3
}

enum Value
{
    A = 2, 
    B // 3
}

Structures & References

Structs group related data fields together. While passing composite objects into fields directly copies data and may waste memory allocations, prefixing type variables with & borrows them cleanly.

struct and references usage
/! Seperating different type attributes is recommended for readiblity!/
public struct Animal
{
    str name;
    str type;
    i32 age;
}

// works the same without reference but wastes memory!
fn PrintAnimal(&Animal pet)
{
    str name = pet.name;
    str type = pet.type;
    i32 age  = pet.age;
    Printf("Your {type} is called {name} and aged {age}\n");
}

fn UpdateAnimal(&mut Animal pet, str nName, i32 nAge) -> bool
{
    if (0 <= nAge <= 100)
    {
        pet.name = nName;
        pet.age  = nAge;
        return true;
    }
    return false;
}

Arrays & Generic Definitions

Arrays utilize explicit sizes at initialization bounds, allocated via the new expression framework. Generic structs allow custom parameters (<T>) to wrap raw array lists safely.

fixed array lists and parameter generic collections
struct Test
{
    str name;
    i32 age;
}

struct Array<T>
{
    T[] elt;
    i32 nbElt;
}

// Example usage context inside logic hooks
fn InitializeStructure()
{
    Test[] content = new Test[5];
    Array<Test> list = { content, 5 };
}
