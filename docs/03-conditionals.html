Conditionals & Control Flow

Branching, loops, range iterations, and structural pattern matching in Nexus 1.3.

Overview

Nexus supports explicit evaluation branching using if blocks, standard collection/interval processing with for : range expressions, while loops, and structural match mechanics.

Chained Relations: Instead of relying on missing logical boolean operators like &&, Nexus 1.3 cleanly evaluates chained mathematical constraints directly, such as if (0 <= age <= 100).

If / Else If / Else

Conditions require surrounding parentheses. Braces are mandatory for multi-statement scopes, though explicitly using them everywhere maintains code predictability.

conditional branching with chained limits
public fn CreateAnimal(str name, str type, i32 age) -> Option<Animal>
{
    // Clean chained evaluation bounds checking
    if (0 <= age <= 100)
    {
        Animal ani = {name, type, age};
        return Option.Some(ani);
    }
    return Option.None;
}

Loops & Iteration

Nexus handles loops through two methods: flexible state evaluations via while, or continuous indexing values via for : range() bounds configurations.

while and range loop structures
// Safe evaluation sequence using while mechanics
while (start > end) {
    i32 tmp = start;
    start = end;
    end = tmp;
}

// Seamless array loops processing via explicit interval indexes
public fn CreateListAnimal() -> Option<Animal>[]
{
    Option<Animal>[] lst = new Option<Animal>[5];
    for (i32 i : range(5))
    {
        lst[i] = CreateAnimal("Bob", "Cat", RandomInt(-5, 4));
    }
    return lst;
}

Pattern Matching

The match block unwraps compound structural states or raw enum variants. Destructured parameters inside matching blocks isolate target variables locally without leaking context scopes.

matching variants and extraction syntax
for (Option<Animal> a : lst)
{
    match (a)
    {
        Option.Some(animal) => {
            str test = animal.name;
            Printf("Test {test}\n");
        }
        Option.None => {
            Printf("#FF0000Not a valid animal\n");
        }
    }
}
